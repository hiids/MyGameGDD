# MyGameGDD
本人为中科大在读大二学生，专业为中科大少年班学院人工智能方向，有兴趣尝试获得策划方向（特别是战斗策划或者普通执行策划）的offer。大二下学期的课业不算多，所以寒假和下学期都有充足的时间来参与策划岗位的实习。
本人技能包括：C#、基础的blender、unity、编剧能力和游戏文案撰写能力、游戏设计、英语四六级等。

该文档中的大一下简历是我做了滚动的天空第一关的复刻的项目和一个打砖块小游戏，尝试投递简历，被四川的独立游戏工作室面试后，由于学不到很多东西，我单方面拒绝了这个机会。后来因为电脑内存不够，加上这种极小的游戏算不上客户端方向的项目，于是全部删掉了。之后相比于客户端方向，更希望冲击策划方向（尤其是战斗策划）。

该文档中的其余内容是我目前自行创作游戏《九九归一》的设计文档，分为总策划案（设计目的、平台和受众、全流程梗概等）、文案设计（5w字剧本+世界观+角色设计+游戏副剧情）、战斗策划（角色和敌人设计、科技树设计、地图设计、升级节点设计）和代码设计等。目前，文档依旧处于未完全定稿阶段。《九九归一》是一款 2d 横屏 Roguelite 类吸血鬼幸存者游戏（类似游戏有：黎明前20分钟、brotato、吸血鬼幸存者等），玩家可以构建强力技能组合，通过对抗浪潮般的敌人和进行深度的策略构筑，体验‘反抗命运’与‘接受终局’的核心叙事张力。

本人于去年12月参与了腾讯青科实训营的活动，线上听腾讯在职的策划人员的课程，并获得证书。

有关游戏设计，本人比较喜欢看美国一年一度的GDC、GameRes游资网、机核等网站，从中学到的东西整理成typora文档，相当实用。本人线下游玩非对称竞技《第五人格》、格斗游戏《街霸6》、fps游戏《无畏契约》、解谜游戏《纸嫁衣》《南瓜先生大冒险》、乙女游戏《恋与深空》及各类steam中的单机游戏（包括且不限于《荒野大表哥2：救赎》《幽灵行者1/2》《锈湖》系列）等，对于设计任何类别的游戏都有信心。
